# Lumark Project Showcase

This folder contains a comprehensive archive of the Lumark project's development and current state.

## Folder Structure

- **branding/**: Contains the official Lumark logo and favicon images.
- **data/**: Contains the current running data in JSON format (document vectors, runs, and processed images).
- **process/**: contains screenshots (`.png`) and browser recordings (`.webp`) showcasing the making process:
  - `zynapse_verification...`: Initial verification of the base platform.
  - `lumark_rebranding...`: The process of applying the Lumark identity.
  - `lumark_design_refinement...`: Moving the logo to the top-left and restoring the premium header.
  - `fix_chart_visibility...`: Resolving contrast issues for the analytical charts.
  - `statistical_trend_intelligence...`: Successful verification of high-contrast metrics.

## Key Documents
- **[LUMARK_NAME.md](../LUMARK_NAME.md)**: Explains the strategy and meaning behind the Lumark name.
- **[walkthrough.md](./walkthrough.md)**: A detailed summary of all changes made during this session, including verification results.
